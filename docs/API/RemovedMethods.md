# (Re-)Moved Methods


## Register new Quest Types

When you want to register new Conditions, Events, (Objectives) or Variables you need to use the [new System] 
which you can access with the respective `QuestTypeRegistry`, which you can get
with `BetonQuest.getInstance().getQuestRegistries()`.

`ConversationIO`, `Interceptor`, `NotifyIO`, `Schedule` and `QuestItem` will be added soon™ to Registries.
