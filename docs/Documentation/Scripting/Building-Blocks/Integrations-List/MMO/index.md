# MMO

Since questing and conversations are the core of MMOs, there are a few MMO plugins that BetonQuest integrates with.

## Provided Integrations

BetonQuest provides integrations for the following MMO plugins:

- [AuraSkills](./AuraSkills.md)
- [Heroes](./Heroes.md)
- [Magic](./Magic.md)
- [mcMMO](./McMMO.md)
- [MMOCore](./MMOCore.md)
- [MMOLib](./MMOLib.md)
- [Fabled](./Fabled.md)
