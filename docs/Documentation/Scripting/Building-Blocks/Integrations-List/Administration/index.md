# Administration

There are a number of common plugins used to manage the server.
Those offer structure and functionality that you might want to use in your quests.

## Provided Integrations

BetonQuest provides integrations for the following administration plugins:

- [LuckPerms](./LuckPerms.md)
- [PacketEvents](./PacketEvents.md)
- [PlaceholderAPI](./PlaceholderAPI.md)
- [RedisChat](./RedisChat.md)
- [WorldEdit & FastAsyncWorldEdit](./WorldEdit.md)
- [WorldGuard](./WorldGuard.md)
