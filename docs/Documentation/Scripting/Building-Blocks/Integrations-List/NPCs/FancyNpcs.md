# [FancyNpcs](https://modrinth.com/plugin/fancynpcs)

!!! info ""
    **Required FancyNpcs version: _2.4.2_ or above, but below _3.0.0_**

If you have this plugin you can use its NPCs.
It is more lightweight than [Citizens](./Citizens.md) but lack some of its features.

!!! info
    FancyNpcs integration supports all [BetonQuest NPC](./index.md) features.
